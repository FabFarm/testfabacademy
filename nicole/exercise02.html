6. The "MANTRA"

git pull
git add --all
git commit -m yourmessagehere
git push

This series of commands would make sure:

  1. first (git pull) my git was updated with my origin
  2. second you add all new alteration to my local branch (git add--alls)
  3. third I committed all changes (git commit -m yourmessagehere)
  4. and last I sent the commits to my origin with(git push)
